# node-work

## 설치 및 구성

```bash
# 1. 프로젝트 생성
git clone git@github.com:JHyeok/node-work.git

# 2. 프로젝트 폴더로 이동
cd node-work

# 3. 의존성 설치
npm install
```

## 실행

```bash
$ npm run start
```
